      Remarks

      This base class acts on incoming measurements, in a non-time-aligned fashion, for general processing. If derived class needs time-aligned data for processing, the ActionAdapterBase class should be used instead. Derived classes are expected call OnNewMeasurements(ICollectionIMeasurement) for any new measurements that may get created.
