      Remarks

      This base class acts as a measurement concentrator which time aligns all incoming measurements for proper processing. Derived classes are expected to override PublishFrame(IFrame, Int32) to handle time aligned measurements and call OnNewMeasurements(ICollectionIMeasurement) for any new measurements that may get created.
